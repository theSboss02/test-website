# Selina Lechner – Physiotherapie (One‑Page)

## Struktur
- index.html
- impressum.html
- datenschutz.html
- css/style.css
- css/fonts.css
- img/selina.svg (Platzhalter – ersetze durch dein Portrait, z.B. img/selina.jpg und ändere src)

## Lokal testen
Im Projektordner:
- `python -m http.server 8000`
Dann öffnen: http://localhost:8000

## GitHub Pages
1. Repository erstellen (z.B. `username.github.io` oder ein Projekt-Repo)
2. Dateien hochladen
3. Settings → Pages aktivieren
